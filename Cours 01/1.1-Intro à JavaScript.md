# Introduction à JavaScript

## 1. Exécuter JavaScript

JavaScript est un langage de programmation. À l'origine, c'était un langage **interprété**, mais aujourd'hui, on parle plutôt d'une **exécution via une compilation JIT (Just In Time)**. Donc il n'est ni 100% interprété, ni 100% compilé.

De nos jours, on utilise des **moteurs JavaScript** pour l'exécution du code JS. Les navigateurs modernes fournissent évidemment un moteur JavaScript :

|Navigateur|Moteur JS|
|---|---|
|Chrome|V8|
|FireFox|SpiderMonkey|
|Safari|JavaScriptCore|

Le navigateur utilise un moteur JS ainsi que des APIs Web telles que le `DOM` (Document Object Model), `fetch` (pour récupérer des données d'un serveur), `Web Storage`, etc.

Cependant, il existe une manière différente d'exécuter JavaScript sans nécessiter un navigateur : `Node.js`, qui est composé des éléments suivants :

- Le moteur **V8** de Chrome
- Une couche de C++
- Des APIs système

**Résultat** : JavaScript peut interagir avec la machine. À condition bien entendu que Node.js soit installé sur ladite machine!

### Différence navigateur vs Node.js (tableau comparatif)

|Critère|Navigateur|Node.js|
|---|---|---|
|Moteur JS|Oui (variable)|Oui (V8 de Chrome)|
|DOM|✅|❌|
|Accès aux fichiers|❌|✅|
|Réseau|Limité|✅|
|Objectif|Interface utilisateur|Serveur / scripts|
|Environnement|*Sandbox* (isolé et sécuritaire)|Système|
