
# Procédure de base : `npm init`

Cette courte procédure explique comment utiliser la commande `npm init` pour **créer un nouveau projet Node.js**.

## 1. Pré-requis

Avant d'utiliser `npm init`, assurez-vous que :

- **Node.js** est installé
- **`npm`** (inclus avec Node.js) fonctionne

Vous pouvez vérifier avec :

```bash
node -v
npm -v
```

## 2. Création d'un nouveau projet

1. Créez un dossier pour votre projet :

    ```bash
    mkdir mon-projet
    cd mon-projet
    ```

2. Lancez l'initialisation du projet :

    ```bash
    npm init
    ```

3. Répondez aux questions (nom du projet, version, description, entrée principale, etc.).

4. À la fin, un fichier **`package.json`** est généré automatiquement.

## 3. Initialisation rapide

Si vous voulez créer un fichier `package.json` avec les valeurs par défaut :

```bash
npm init -y
```

## 4. Résultat : `package.json`

Ce fichier contient les informations de configuration du projet, comme :

- le nom,
- la version,
- les dépendances,
- les scripts.

Il est essentiel pour gérer les bibliothèques et automatiser des tâches avec `npm`.

## 5. Exemple simple de `package.json`

```json
{
  "name": "mon-projet",
  "version": "1.0.0",
  "description": "Exemple de projet Node.js",
  "main": "index.js",
  "scripts": {
    "start": "node index.js"
  }
}
```
